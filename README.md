# Developer Academy

A private 60-day coding academy focused on rebuilding JavaScript, React, and AI development skills.

## Core Features
- Daily lessons (60 days)
- Fill-in-the-blank quizzes
- Multiple-choice quizzes
- Coding challenge each day
- Explanations for every answer
- Progress tracking
- Mastery tracking
- Streaks
- Notes
- LocalStorage persistence

This project is intended for personal use and learning.
